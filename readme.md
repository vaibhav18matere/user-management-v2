
# User Management

A simple user management web app where user can register, log in and add details to showcase to dashboard


## Tech Stack

**Client:** React

**Server:** Node, Express

**Database:** Mongo DB


## Features

- New User Registration
- Login using credentials
- Update data and user information fields
